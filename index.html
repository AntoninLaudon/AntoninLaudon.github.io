Introduction

L’Arduino

L’Arduino, qu’est-ce que c’est ?

Une carte Arduino est une carte équipée d’un micro-contrôleur, d’entrées et de sorties.
Son micro-contrôleur est capable de lancer un programme écrit par l’utilisateur afin de gérer les entrées et les sorties de la carte.

Au programme

Qu’allons-nous faire dans l’heure qui suit ?

Afin de découvrir l’Arduino, nous allons effectuer un montage électronique et un programme simple qui formeront un jeu de réflexe.

Règles du jeu

Le but du jeu est très simple, il faut être le premier à buzzer dès lors qu’un témoin lumineux s’allume.

3. Allumer et éteindre une LED et temporiser

void loop() {
	digitalWrite(pin::led2, HIGH);
	delay(1000);
	digitalWrite(pin::led2, LOW);
	delay(1000);
}

Dans cet exemple, on utilise l’appel digitalWrite(pin::led2, HIGH) pour allumer notre LED numéro 2. L’appel digitalWrite(pin::led2, LOW) permet d’éteindre la LED.

Il est possible de remplacer pin::led2 par une des autres LED, comme pin::led3 par exemple.

On remarque dans l’exemple un autre appel, delay(1000). Il a pour effet de faire attendre l’Arduino pendant 1000 millisecondes, soit une seconde. L’utilisation de delay, notamment dans une boucle, permet d’éviter de faire surchauffer la carte.

Pour tester le code : appuyez sur le bouton rond contenant une flèche pointant vers la droite. Cela va envoyer votre programme à la carte.

4. Commutateur

Nous allons vous donner un bout de code que vous allez mettre à la place de la fonction void loop() et de son contenu.

Avant de rajouter l’exemple qui suit dans votre programme, effacez la fonction void loop() déjà présente. Le contenu de la fonction qui est entre accolades, doit aussi être supprimé.

void loop() {
	if (pin::state1 == HIGH) {
		// Si le premier bouton a été activé, on allume la LED 2.
		digitalWrite(pin::led2, HIGH);
		// On n'oublie pas de remettre l'état du bouton à 0.
		pin::state1 = LOW;
	} else if (pin::state2 == HIGH) {
		// Si le deuxième bouton à été activé, on éteint la LED susmentionnée.
		digitalWrite(pin::led2, LOW);
		// On n'oublie pas de remettre l'état du bouton à 0.
		pin::state2 = LOW;
	}
	// Le délai permet d'éviter la chauffe inutile de la carte.
	delay(10);
}

Même chose qu’avant, pour tester, il suffit d’appuyer sur le bouton de téléchargement en amont vers la carte (bouton rond avec une flèche qui pointe vers la gauche)

5. Début du jeu

Nous allons ajouter une fonction qui a pour but d’attendre que l’on appuie sur les deux boutons avant de lancer une partie.

Le code à ajouter, juste avant void loop() :

/**
 *  Permet de savoir quand est-ce que le temps imparti est écoulé.
 */
unsigned long timerEnd(0);

/**
 * Cette fonction doit attendre du moment que la partie n'a pas commencé.
 */
void waitStart() {
	// Tant que les deux boutons n'ont pas été pressés, on attend 10ms...
	while (pin::state1 != HIGH || pin::state2 != HIGH) {
		digitalWrite(pin::led1, pin::state1);
		digitalWrite(pin::led3, pin::state2);
		delay(10);
	}

	// On confirme que la partie va commencer en allumant toutes les
	// LED.
	digitalWrite(pin::led1, HIGH);
	digitalWrite(pin::led2, HIGH);
	digitalWrite(pin::led3, HIGH);

	// On défini aléatoirement le temps à attendre avant que la LED
	// témoin s'allume.
	auto timeToWait(random(2000, 7000));
	timerEnd = millis() + timeToWait;

	// On attend un peu puis on éteint toutes les LED.
	delay(500);
	digitalWrite(pin::led2, LOW);
	digitalWrite(pin::led1, LOW);
	digitalWrite(pin::led3, LOW);

	// On remet les états à 0 pour pouvoir les consulter plus tard.
	pin::state1 = LOW;
	pin::state2 = LOW;
}

Pour appeler la fonction, il suffit de remplacer la fonction void loop() par :

void loop() {
	// On appelle notre nouvelle fonction :
	waitStart();
}

Pour tester le code, il suffit de faire comme au cours des tâches précédentes.

Si tout marche comme prévu, lorsque vous appuyez sur les deux boutons (pas forcément en simultané) toutes les lumières vont s’allumer, puis s’éteindre après une demi-seconde.

Si vous avez un problème, demandez de l’aide à un Cobra !

6. Temps d’attente et défaite prématurée

Maintenant, il faut que l’on ajoute un mécanisme qui attend que le temps imparti soit écoulé. Si un joueur appuie sur son buzzer avant que la lumière se soit allumée, il faut donner le point à son adversaire.

Pour ce faire on rajoute les fonctions suivantes au code juste sous void loop() :

/**
 * Cette fonction est appelée lorsque le joueur 1 a gagné
 */
void winP1() {
	digitalWrite(pin::led1, HIGH);
	digitalWrite(pin::led3, LOW);
}

/**
 * Cette fonction est appelée lorsque le joueur 2 a gagné
 */
void winP2() {
	digitalWrite(pin::led3, HIGH);
	digitalWrite(pin::led1, LOW);
}

/**
 * Cette fonction attend que le timer soit écoulé ou qu'un joueur se soit
 * brûlé les ailes.
 */
bool waitTimer() {
	while (millis() < timerEnd) {
		if (pin::state1 == HIGH) {
			pin::state1 = LOW;
			winP2();
			return false;
		} else if (pin::state2 == HIGH) {
			pin::state2 = LOW;
			winP1();
			return false;
		}
		delay(10);
	}
	digitalWrite(pin::led2, HIGH);
	return true;
}

On modifie également le contenu de void loop() pour qu’il soit comme ceci :

void loop() {
	waitStart();
	if (waitTimer() == true) {
		// C'est ici que la partie va continuer si personne n'a
		// perdu pendant l'attente.
	}

	// A la fin de la partie, on attend 2 secondes puis on éteint le
	// témoin et on remet les états des boutons à 0.
	delay(2000);
	pin::state1 = LOW;
	pin::state2 = LOW;
	digitalWrite(pin::led2, LOW);
}

Vous pouvez tester de la même façon qu’avant.

Pour vous assurer du fonctionnement du programme, après l’avoir téléchargé sur la carte, il suffit de lancer la partie et d’appuyer rapidement sur un des boutons. La LED opposée devrait s’allumer pour montrer que le joueur qui n’a pas appuyé a gagné.

7. Victoire en conditions normales

Maintenant, une fois que le timer est écoulé, il faut être en mesure de faire gagner le joueur qui buzzera le plus rapidement.

Pour ce faire, on va rajouter la fonction suivante :

/**
 *  Cette fonction attend qu'un joueur déclenche son buzzer et le fait gagner.
 */
void waitTrig() {
	while (!pin::state1 && !pin::state2)
		delay(10);
	if (pin::state1 == HIGH)
		winP1();
	else if (pin::state2 == HIGH)
		winP2();
}

N’oubliez pas de modifier la fonction void loop() afin de la faire employer notre nouvelle fonction :

void loop() {
	waitStart();
	if (waitTimer() == true) {
		waitTrig();
	}

	// A la fin de la partie, on attend 2 secondes puis on éteint le
	// témoin et on remet les états des boutons à 0.
	delay(2000);
	pin::state1 = LOW;
	pin::state2 = LOW;
	digitalWrite(pin::led2, LOW);
}

Maintenant, vous pouvez mesurer vos réflexes avec vos binômes et avec les autres groupes !

N’oubliez pas de télécharger le nouveau programme vers la carte.
